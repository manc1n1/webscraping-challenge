## Installation

1.  Clone the repo

    ```sh
    git clone https://github.com/manc1n1/webscraping-challenge.git
    ```

2.  Change directories to `webscraping-challenge`

    ```sh
    cd webscraping-challenge
    ```

3.  Run `part_1_mars_news.ipynb` and `part_2_mars_weather.ipynb`
